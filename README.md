# Investment Landing Page

A modern, responsive landing page designed for a **sustainable investment company**, focusing on clean layout, usability, and performance. This project showcases strong 
fundamentals in **semantic HTML**, **modern CSS layout techniques**, and **Vanilla JavaScript** for interactivity. 

The landing page consists of a **single-page layout (home page only)**, emphasizing clarity, accessibility, and a professional visual identity aligned with sustainability 
and long-term investment values.

## Key Features

* **Responsive Layout**
  Fully responsive design that adapts smoothly across desktops, tablets, and mobile devices.

* **Flexbox-Based Layout Control**
  Flexbox is used extensively to manage spacing, alignment, and container behavior in a clean and predictable way.

* **CSS Grid for Column Structure**
  Grid Layout is applied to create well-structured columns and sections, ensuring visual balance and scalability.

* **Modern & Clean UI**
  Minimalist and contemporary design, suitable for professional investment and sustainability-focused brands.

* **Responsive Animated Navigation Menu**
  A mobile-friendly menu built with **Vanilla JavaScript**, featuring smooth animations and intuitive interactions.

* **Pure HTML & CSS Accordion**
  Accordion component implemented **without JavaScript**, using only `input` + `label` for state control and accessibility.
